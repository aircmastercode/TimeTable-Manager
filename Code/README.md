# Timetable Generator — Search

University timetable as **state-space search** (Project 4, L3 formulation). This repo is being built in layers: problem model → costs/heuristics → search → plots → CLI.

## Status

Skeleton phase: dependencies and project metadata are in place; domain and search code are added in later commits (see team playbook).

## Setup

Python **3.10+** recommended.

```bash
cd Code
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

Run instructions, CLI flags, and output file descriptions are documented in the **full README** once `main.py` and the rest of the stack exist (final documentation commit).
